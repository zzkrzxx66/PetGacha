# 🐾 PetGacha - 宠物精灵抽卡游戏

一款可爱的宠物精灵抽卡游戏，收集50只不同稀有度的宠物精灵！

## ✨ 功能特色

- 🎴 **抽卡系统** - 单抽/十连，保底机制
- 📖 **宠物图鉴** - 收集进度展示
- 👥 **阵容系统** - 组建3人小队
- 💰 **经济系统** - 金币管理
- 🎨 **精美UI** - Material 3 设计

## 🎮 游戏玩法

### 抽卡
- **单抽**: 100金币
- **十连**: 900金币（保底SR+）
- **保底**: 50抽内必出SSR+

### 稀有度
| 稀有度 | 概率 | 颜色 |
|--------|------|------|
| N 普通 | 50% | ⚪ 灰色 |
| R 稀有 | 28% | 🟢 绿色 |
| SR 史诗 | 15% | 🔵 蓝色 |
| SSR 传说 | 6% | 🟣 紫色 |
| UR 神话 | 1% | 🟡 金色 |

## 🛠️ 技术栈

- **语言**: Kotlin
- **UI**: Jetpack Compose + Material 3
- **架构**: MVVM
- **数据库**: Room (SQLite)
- **导航**: Jetpack Navigation
- **编译**: GitHub Actions

## 📦 构建

1. 克隆项目
```bash
git clone https://github.com/你的用户名/PetGacha.git
```

2. 使用 Android Studio 打开项目

3. 运行 `./gradlew assembleDebug` 构建APK
